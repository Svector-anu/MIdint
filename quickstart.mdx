## Get Your DEX Running

Follow these steps to deploy a complete DEX on Bitcoin in under 20 minutes.

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/Svector-anu/midint.git
    cd midint/contracts
    pnpm install
    ```
  </Step>

  <Step title="Configure Your Wallet">
    Create `.env` file:
    ```bash
    cp .env.example .env
    ```
    
    Add your test wallet mnemonic:
    ```bash
    MNEMONIC="your twelve word test mnemonic here"
    RPC_URL="https://rpc.regtest.midl.xyz"
    ```
    
    <Warning>
      **Use a TEST wallet only!** Never use your real Bitcoin wallet for development.
    </Warning>
  </Step>

  <Step title="Deploy All Contracts">
    ```bash
    npx hardhat deploy --network default
    ```
    
    This deploys:
    - âœ… TBTC & WBTC tokens
    - âœ… Uniswap V2 Factory
    - âœ… Uniswap V2 Router
    - âœ… TBTC/WBTC trading pair
    - âœ… Initial liquidity
  </Step>

  <Step title="Mint Test Tokens">
    You can get test tokens in two ways:
    
    **Option A: Web Faucet (Recommended)**
    Visit [https://faucet.regtest.midl.xyz/](https://faucet.regtest.midl.xyz/) and enter your Bitcoin address.
    
    **Option B: Hardhat Script**
    ```bash
    npx hardhat deploy --tags Faucet --network default
    ```

  </Step>

  <Step title="Test Your DEX">
    ```bash
    npx hardhat deploy --tags TestDex --network default
    ```
    
    This will:

    - âœ… Add liquidity (100 TBTC + 1 WBTC)

  </Step>
</Steps>
