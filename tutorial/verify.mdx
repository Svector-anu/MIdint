## The Source of Truth

Because MIDL is EVM-compatible, we can use traditional blockchain explorers like **Blockscout** to see everything happening on Bitcoin L1.

### 1. Open Explorer

Navigate to the Regtest Explorer:
[blockscout.regtest.midl.xyz](https://blockscout.regtest.midl.xyz)

### 2. Search Your Address

Paste your EVM address into the search bar. You should see:

<img src="/docs/screenshots/03_blockscout_transactions.png" alt="Blockscout Address View" />

### 3. Check These Items:

<CardGroup cols={2}>
  <Card title="Token Balances" icon="wallet">
    Verify you have TBTC, WBTC, and LP tokens in your wallet.
  </Card>
  <Card title="Transaction History" icon="list">
    See the "Swap" and "Add Liquidity" transactions confirmed.
  </Card>
  <Card title="Contract Source" icon="code">
    Interact with the Router or Factory directly from the UI.
  </Card>
  <Card title="Gas Fees" icon="gas-pump">
    See the Bitcoin fees used for each operation.
  </Card>
</CardGroup>
