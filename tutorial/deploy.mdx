## Launching the DEX

Now we're going to deploy a complete Uniswap V2 infrastructure to Bitcoin. This includes the Factory, the Router, and our trading pair.

### Execute Deployment

Run the follow command to deploy everything:

```bash
npx hardhat deploy --network default
```

### What's happening?

<AccordionGroup>
  <Accordion icon="coins" title="1. Token Deployment">
    We deploy **TBTC** (Test BTC) and **WBTC** (Wrapped BTC) to use inside our DEX.
  </Accordion>
  <Accordion icon="factory" title="2. Uniswap Factory">
    The engine that creates and manages trading pairs.
  </Accordion>
  <Accordion icon="route" title="3. Uniswap Router">
    The contract that handles all your swaps and liquidity provision.
  </Accordion>
  <Accordion icon="droplet" title="4. Pair & Liquidity">
    We create the TBTC/WBTC pair and add the first bit of liquidity so others can trade!
  </Accordion>
</AccordionGroup>

### View Transaction logs

Your terminal will show the contract addresses as they are deployed. 

<Check>
  Once complete, your DEX is live on Bitcoin!
